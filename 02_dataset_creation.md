# Step 2 - Emotional Dataset Creation

## Dataset Format

```json
{
  "text": "I feel lonely today",
  "emotion": "sad"
}
```

## Emotions
- happy
- sad
- angry
- stress
- anxiety
- lonely
- neutral

## Dataset Sources
- Reddit
- Mental health conversations
- Therapy examples
- Your own generated conversations

## Useful Public Datasets
- GoEmotions
- Emotion Dataset
- Empathetic Dialogues

## Folder Structure

```text
dataset/
    train.json
    test.json
    validation.json
```
