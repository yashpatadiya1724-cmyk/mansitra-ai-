# Manasitra Emotional AI - Project Overview

## Goal
Build a fully custom emotional AI assistant for the Manasitra app without external APIs.

## Features
- Emotion Detection
- Memory System
- Emotional Chatbot
- Voice Assistant
- Mood Tracking
- Crisis Detection
- Multilingual Support

## Main Architecture

```text
User Input
    ↓
Emotion Detection
    ↓
Memory Engine
    ↓
Conversation Brain
    ↓
Response Generator
    ↓
Voice + UI
```

## Recommended Stack

### Frontend
- React
- Vite
- Tailwind CSS

### Backend
- Python FastAPI

### AI
- PyTorch
- Transformers

### Database
- PostgreSQL / MongoDB
